# AI Therapist Frontend

## Setup

1. Install dependencies:
   ```sh
   npm install
   # or
   yarn
   ```

2. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```

The app will be available at http://localhost:5173 and will proxy API requests to your Flask backend at http://localhost:5000.

---

- Hold the green button to record, release to send your voice to the agent.
- The agent's response will play automatically.
